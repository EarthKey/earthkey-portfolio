## 1. ritsuto.org（リツト公式サイト）

**URL**: https://ritsuto.org/#/
**属性**: Antigravity教科書著者 / AIボイスクリエイター / AI活用講師 / MC / Chef

### セクション構成（上から）

```
Hero          ← キャラ画像 + 役割バッジ（教科書著者・司会者・AI活用講師）
About         ← リツトについて（短文）
PRODUCTS      ← 商品2点（教科書・ロードマップ）
AI Voice      ← AIボイス「kuon」専用セクション
AI Programming ← プログラミング講座
Master of Ceremonies ← MC活動
Event Producer ← イベント企画
Voice Works   ← 音声制作実績
Chef          ← 料理（個人の多面性）
STORY         ← 発信の背景・思い（ナラティブ）
CTA           ← 「もっとリツトを知る」
```

### 参考にすべき点

| 要素 | 内容 |
|---|---|
| **役割バッジ** | Heroに「教科書著者」「AI活用講師」など役割を視覚的ラベルで表示 |
| **セクション英語見出し** | PRODUCTS / AI Voice / STORY など英語で統一 → スキャンしやすい |
| **活動ごとに独立セクション** | 一覧ではなく各活動を「フィーチャーセクション」として独立させている |
| **STORYセクション** | ページ末尾に「なぜこれをやっているか」の個人ナラティブを配置 |
| **キャラクターブランディング** | AIボイス「kuon」のキャラで覚えてもらう仕組み |
| **各セクションにCTA** | 「詳しくはこちら」「購入する」を各セクションに配置 |

## 2. kishino-works.jimdofree.com（kishino Art works）

**URL**: https://kishino-works.jimdofree.com/
**属性**: イラストレーター / グッズ制作 / イベント出展 / NFTクリエイター

### ナビゲーション構成

```
TOP / PROFILE / SERVICE / PRICING / PORTFOLIO / SPECIAL / SHOPPING / FAQ / CONTACT
```

### セクション構成（上から）

```
Hero          ← 大判イラスト
近況          ← ブログ的な更新情報（常時更新）
活動スケジュール ← 今後のイベント出展予定
創作活動履歴   ← 過去の実績リスト
SNS・連絡先   ← LINE QRコード + 各SNS
```

### 参考にすべき点

| 要素 | 内容 |
|---|---|
| **近況セクション** | 更新情報を常に上に置くことでリピーター獲得・サイトを「生きている」状態に保つ |
| **PRICING ページ** | 料金を独立ページで明示 → 問い合わせのハードルを下げる |
| **創作活動履歴** | 実績を時系列で積み上げ → 信頼の可視化 |
| **明確なナビ** | SERVICE / PRICING / PORTFOLIO を分けることで目的別に見られる |
| **FAQ** | よくある質問で問い合わせ前のハードルを下げる |
